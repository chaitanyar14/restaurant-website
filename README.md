This project is a multi-page restaurant website developed as part of a Human Computer Interaction (HCI) academic project.

The main objective of this project was to design a user-friendly, visually appealing, and accessible interface by applying HCI principles and frontend web technologies.

---

## Project Overview

The website represents an informational restaurant website where users can easily navigate through different sections such as menu, offers, and contact details.

Special focus was given to usability, clear navigation, and consistent design across all pages.

---

## Pages Implemented

The website consists of the following 5 pages:

1. Home Page (`index.html`)
2. Menu Page (`menu.html`)
3. About Page (`about.html`)
4. Special Offers Page (`offers.html`)
5. Contact Page (`contact.html`)

All pages are connected using a common navigation menu to ensure smooth user flow.

---

## Features

- Multi-page static restaurant website
- Dropdown navigation menu for menu categories
- Responsive and clean UI design
- Consistent layout and color theme
- User-friendly navigation based on HCI principles

---

## Technologies Used

- HTML
- CSS
- JavaScript (basic interactivity)

---

## HCI Concepts Applied

- Usability and learnability
- Consistency in navigation and layout
- Visual hierarchy using headings, colors, and spacing
- Accessibility considerations such as readable fonts and contrast
- Simple and intuitive user flow

---

## Design Details

- Common header and footer across all pages
- Dropdown menu for better content organization
- Clear callouts for offers and menu items
- Focus on minimal cognitive load for users

---

## Project Scope

This project focuses only on frontend design and user interface development.

No backend functionality, database, or real-time ordering system is implemented.  
The website is intended for learning and academic demonstration purposes.

---

## Project Status

Completed as part of an HCI academic course project.
